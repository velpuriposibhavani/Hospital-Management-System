
# 🏥 Hospital Management System in C

## 📌 Description

The **Hospital Management System** is a simple console-based application developed using the **C programming language**. This project helps manage hospital patient records efficiently through a menu-driven interface.

The system allows users to **add, display, and search patient records**. All patient details such as **Patient ID, Name, Age, and Disease** are stored in a file using file handling, so the data can be saved permanently and accessed later.

This project demonstrates fundamental C programming concepts like **structures, functions, file handling, and menu-driven programming**.

---

# 🚀 Features

* Add new patient records
* Display all patient records
* Search patient by ID
* Store patient data in file
* Simple menu-driven interface
